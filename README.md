# hotel-booking-site
Моя первая вёрстка landing page сайта для бронирования отелей с учётом таких особенностей, как: 
- семантика разметки, 
- подключение шрифтов, 
- pixel perfect вёрстка, 
- отправка форм на {JSON} Placeholder, 
- hover-эффекты для кнопок.
